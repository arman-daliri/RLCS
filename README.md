# Article
**RLCS: A Reinforcement Learning based Cluster Selector**

## How to Run

The implementation of the feature selection method is provided in the `RLCS.ipynb` notebook.

First, install the required packages by running:
```sh
pip install -r requirements.txt
```
Next, download the datasets from this [link](https://drive.google.com/drive/folders/1r8wxFnawMCkXZlN0dIZPYQQ8TNIMk8ry) and save them in the project folder.

Finally, run the `RLCS.ipynb` notebook cell by cell.
